## Variant: Hacker Terminal

### Design stance
Web3Privacy aesthetic — pure black canvas, glass-morphism cards, achromatic color system. The "security researcher's terminal" feel: minimal, functional, zero color distraction.

### Key choices
- **Layout:** Dual-column hero (left text, right preview card), 5-col steps, 3-col features
- **Typography:** Inter 700 bold headlines, JetBrains Mono for data. Aggressive letter-spacing -2px at display size
- **Color:** Pure `#030303` background. White text + white CTA. No accent color — everything is grayscale
- **Cards:** `rgba(255,255,255,0.02)` bg, `rgba(255,255,255,0.06)` border. Hover → brighter border + lift
- **CTA:** Stark white button `#fff` on black — maximum contrast, no color
- **Interaction:** Grid background pattern, hover-lift on all cards, filter bar with active state

### Trade-offs
- **Strong at:** Focus, readability, hacker credibility, zero visual noise
- **Weak at:** Emotional warmth, brand differentiation, visual hierarchy (everything is equally muted)

### Best for
Crypto-native users who want function over form. Privacy/security audience. When you want to look like a serious protocol, not a consumer app.
