## Variant: Linear Dark

### Design stance
Linear-inspired precision — near-black canvas with indigo-violet accent as the sole chromatic color. Darkness as the native medium, where hierarchy is managed through luminance gradation, not color.

### Key choices
- **Layout:** Same dual-column hero, identical content structure to Terminal variant
- **Typography:** Inter with OpenType features, weight 510 as signature. Negative letter-spacing -1.584px at display size (tighter than Terminal's -2px)
- **Color:** `#08090a` background (slightly warmer than pure black). Brand indigo `#5e6ad2` for CTA + accent. `#f7f8f8` primary text (not pure white)
- **Cards:** `rgba(255,255,255,0.02-0.05)` bg with luminance stacking. Indigo-tinted hover borders
- **CTA:** Brand indigo `#5e6ad2` primary button. Ghost button for secondary. Color creates clear action hierarchy
- **Interaction:** Ambient indigo glow at top, preview card has indigo glow orb, filter active state uses indigo tint

### Trade-offs
- **Strong at:** Visual hierarchy (color guides attention), brand identity, polish, feels like a product
- **Weak at:** Less "raw" for crypto purists, indigo might feel too mainstream for Web3

### Best for
Teams building a polished product, not just a protocol. When you want to attract users beyond the crypto-native bubble. When brand identity matters.
